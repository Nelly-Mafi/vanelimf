What is a Raspberry Pi?

The Raspberry Pi is a functional single-board computer (SBC) introduced by the Raspberry Pi Foundation based in Cambridge, United Kingdom. The Raspberry Pi integrates the CPU processor and GPU on a single Integrated Circuit (IC). Other functional components include RAM, GPIO Pins, USB and HDMI ports, Bluetooth, Wi-Fi, and other elements soldered on the board. OS operating system installed on a microSD card will be required to store the files. This Raspberry Pi computer connects to a monitor or TV through the HDMI port and uses a keyboard and a mouse to control the Linux operating system. GPIO Pins can connect the Raspberry Pi to various sets of sensors. Astro Pi unit in the International Space Station (ISS) uses The Sense HAT add-on board. Different families and models of Raspberry Pi computers have been released. The device was produced to help people of all ages to improve the coding skills and to create innovative projects.

Raspberry Pi vs Arduino


The Raspberry Pi is a single-board computer microprocessor, while the Arduino is a microcontroller board. The Raspberry Pi runs on a Linux OS. The digital GPIO Pins enable this device to process data efficiently. But the Arduino can control both inputs and outputs using the digital and analog GPIO Pins on its board. The Arduino as a microcontroller (which is a part of a computer), is only able to run only one program kept in its onboard storage. Hence, choosing which device to use is based on the project and hardware to be used. If a project requires video/graphic interfaces, or to process data files for computer-dependent applications, Raspberry Pi would be suitable. If a project requires controlling different circuit components (sensors, LCDs, motors), or battery-powered repetitive projects, Arduino would be suitable.

Setting up the Raspberry Pi

You will need to have both the hardware and software ready to boot up your device. The required hardware includes a monitor connected to the Raspberry Pi computer by HDMI, USB keyboard and mouse, power supply, and a microSD card with the operating system software installed on. The power supply needs to be attached after the microSD insertion into the slot and the HDMI and USB connections (so that circuits are initialized completely). The Raspberry Pi OS (also known as the Raspbian) can be installed on the microSD in 4 steps using the Raspberry Pi Imager (if the kit does not already have a microSD storing the Raspberry Pi OS installer called NOOBS).

By connecting the Raspberry Pi’s USB power port to the power supply, a red light will appear on the board meaning that the device is booting up and the desktop will appear on the monitor. For the first time starting your system, you will need to follow the initial setup steps such as setting up your language, Time zone, Wi-Fi, and a password for your Raspberry Pi. Once the setup is complete, put some time to explore the operating system environment as you are only one step away from creating! Keep your Raspberry Pi board protected in a case.

Programming


The Recommended programming language for the Raspberry Pi is Python. Python’s English-like syntax is easy to read, learn, and write. Thonny, a Python3 integrated development environment (IDE) is pre-installed in the Raspberry Pi OS (note that the Arduino IDE is written in a different language). Find and open Thonny under access menu (raspberry symbol) > programming > Thonny Python IDE. After opening Thonny, you will see two main sections: the script editor and the shell. By running the written script in the editor, the shell will execute the program.

Same as learning other programming languages, you will need to learn the syntax and semantics of Python. Have the links below as your Python language reference.

Sense HAT

The Sense HAT is an extension board for Raspberry Pi attached to it via the GPIO pins. The Sense HAT will provide temperature, humidity, pressure, and orientation sensors (integrated circuits) to detect, quantify, and send the data to the processor. Sense HAT also provides a five-button joystick and a built-in 8x8 RGB LED matrix. Explore the links below for setting up your Sense HAT.

The LED matrix can display an image or scroll a text message. Three orientation sensors are available: gyroscope measuring the angular velocity, accelerometer measuring acceleration, and magnetometer measuring the magnetic field. With the Sense HAT, you can perform scientific experiments, and create software applications and games. Sense HAT needs to be installed in the Command-line interface. Installing any Python library (modules) including the Sense HAT library will also happen in the Command-line environment.
You can also explore Python with the Sense HAT using the Online Sense HAT emulator.

Measuring the temperature and humidity of the atmosphere
The purpose of this project is to measure the temperature and humidity of the surrounding environment using the Raspberry Pi and the Sense HAT. This project has also been executed in the International Space Station’s installed Raspberry Pi computer (Astro Pi). The temperature and humidity sensors on the Sense HAT will measure the physical quantities and send the data to Raspberry Pi. The Python code will enable you to select the option that you want to be displayed on the LED matrix display (temperature or humidity). SenseHAT: Part 1 Temperature and humidity

The game is displayed on the 8x8 LED matrix, and you can control the snake with the joystick. In this game, you will need to understand game designing and know the game script structure. Installing Google Assistant on the Raspberry Pi
This project shows how to create Google Assistant with the Raspberry Pi that is useful as a smart speaker, and for playing music. Artificial intelligence will answer all your questions. Google Assistant has the entirety of Google knowledge. Hence, enabling Google application programming interface (API) is required to communicate with Google Services. The Raspberry Pi will need to be connected to a microphone and a speaker. Also, the Raspberry Pi is required to authenticate the activity controls (such as voice and audio activities, and device information) to the API. Google Assistant on Raspberry Pi
